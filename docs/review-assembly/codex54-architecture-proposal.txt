  Net effect

  This preserves the intended flow:

  - pre-compaction agent sees pressure
  - agent silently registers evacuation work
  - compaction happens
  - post-compaction delegates are released from lifecycle state
  - their returns wake the compacted main session through structured trigger metadata
  - the main session gets silent enrichment plus post-compaction context

  So the fundamentals stay the same. The fix is mostly about putting each piece of state in the right layer.
